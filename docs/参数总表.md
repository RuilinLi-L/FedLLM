# FedLLM 参数总表

本文档按当前代码入口整理参数含义，主要覆盖 `attack.py` / `attack_new.py`、`train.py`、`scripts/proxy_utility.py` 以及 baseline wrapper 脚本。默认值以代码为准，baseline sweep 值以 `scripts/*.sh` 为准。

## 1. 入口脚本与实验角色

| 参数 / 脚本 | 默认值 / 取值 | 所属入口 | 含义 |
|---|---:|---|---|
| `attack.py` | - | 攻击评估 | 主 DAGER 文本恢复入口；读取 `args_factory.py` 参数，输出 `RESULT SUMMARY`。 |
| `attack_new.py` | - | 攻击评估 | 旧/并行 DAGER 攻击入口，也复用 `args_factory.py`。当前正式汇总优先看 `attack.py`。 |
| `train.py` | - | 训练 utility | 带防御的端到端微调入口，输出 `TRAIN RESULT SUMMARY`。 |
| `scripts/proxy_utility.py` | - | Proxy utility | 不完整重训，只做 one-step virtual update 的 utility 近似评估，输出 `PROXY UTILITY SUMMARY`。 |
| `scripts/defense_baselines.sh` | - | baseline 隐私实验 | 批量跑 full-gradient DAGER 下的防御 baseline sweep。 |
| `scripts/utility_baselines.sh` | - | baseline utility 实验 | 批量跑 proxy utility + 三 seed 训练 utility，并可合并 privacy-utility tradeoff。 |
| `scripts/proxy_baselines.sh` | - | proxy-only utility | 只跑 proxy utility baseline，不跑训练期 utility。 |
| `scripts/peft_baselines.sh` | - | PEFT 隐私实验 | 批量评估 PEFT/LoRA/IA3 更新泄露与防御 baseline。 |
| `scripts/partial_gradient_baselines.sh` | - | partial-gradient 隐私实验 | 批量评估只暴露部分层/部分参数的泄露。 |
| `scripts/lrb_ablation.sh` | - | LRB 消融 | 批量跑 LRB 变体的 privacy、proxy、train 三类实验。 |
| `scripts/collect_experiment_logs.py` | - | 结果汇总 | 解析日志并导出 raw、utility、privacy-utility tradeoff 表。 |

## 2. 通用日志与复现实验参数

| 参数 | 默认值 / 取值 | 所属入口 | 含义 |
|---|---:|---|---|
| `--rng_seed` | `101` | attack / train / proxy | 随机种子；控制数据抽样、随机防御、训练 shuffle 等。`defense_baselines.sh` 未显式传入时会按 `DAGER_SEEDS` 展开多 seed。 |
| `--neptune` | `None` | attack | Neptune 项目名；为空时不启用。 |
| `--neptune_offline` | `False` | attack | Neptune 离线模式。 |
| `--label` | `name of the run` | attack | Neptune 或日志中的 run 标签。 |
| `--log_file` | `None` | attack / train / proxy | 将 stdout/stderr 镜像写入指定 UTF-8 日志文件。 |
| `--log_append` | `False` | attack / train / proxy | 追加写日志，而不是覆盖。 |
| `DAGER_LOG_DIR` | `log/runs` | wrapper 脚本 | 自动日志根目录。 |
| `DAGER_NO_AUTO_LOG` | unset | wrapper 脚本 | 设置后关闭部分脚本的自动日志目录创建。 |

## 3. 数据、任务与模型参数

| 参数 | 默认值 / 取值 | 所属入口 | 含义 |
|---|---:|---|---|
| `--dataset` | attack 必填；train 默认 `cola` | attack / train / proxy | 数据集。attack 支持 `cola`、`sst2`、`rte`、`rotten_tomatoes`、`stanfordnlp/imdb`、`glnmario/ECHR`；train/proxy 支持 `cola`、`sst2`、`rte`、`rotten_tomatoes`。 |
| `--task` | attack 必填；train/proxy 默认 `seq_class` | attack / train / proxy | 任务类型。主实验为 `seq_class`；attack 还保留 `next_token_pred`。 |
| `--split` | 必填；`val` / `test` | attack | 攻击数据划分。`val` 会跳过前 1000 个样本并按长度分桶抽样；`test` 从前 1000 个样本抽取。 |
| `--pad` | `right`; `right` / `left` | attack | tokenizer padding 方向；GPT/Llama 常用 `left`。 |
| `--batch_size`, `-b` | attack/train 默认 `1`；proxy 默认 `2` | attack / train / proxy | 每个客户端更新或训练 step 的样本数。 |
| `--n_inputs` | 必填 | attack | 攻击评估的 batch/update 数量；实际样本数为 `n_inputs * batch_size`。 |
| `--start_input` | `0` | attack | 从第几个 input batch 开始攻击，便于断点续跑。 |
| `--end_input` | `100000` | attack | 攻击到第几个 input batch 前停止。 |
| `--model_path` | attack/train 默认 `bert-base-uncased`；proxy 必填 | attack / train / proxy | Hugging Face 模型名或本地 checkpoint 路径。 |
| `--finetuned_path` | `None` | attack | 已微调模型或 PEFT adapter 路径；seq_class 正式隐私实验应传入它。 |
| `--tokenizer_path` | `None` | train / proxy | 当本地 checkpoint 缺 tokenizer 文件时，显式指定 tokenizer 来源。 |
| `--cache_dir` | `None` | attack | Hugging Face 缓存目录；wrapper 通常传 `./models_cache`。 |
| `--models_cache` | `./models_cache` | train / proxy | Hugging Face 缓存目录。 |
| `--device` | `cuda` | attack / train / proxy | 主计算设备；`cuda` 会经 `utils.gpu.resolve_cuda_device` 解析。 |
| `--device_grad` | `auto` | attack | 默认跟随 `--device` 解析后的设备；传 `cpu` 可复现旧的 CPU 梯度路径。 |
| `--attn_implementation` | `sdpa`; `sdpa` / `eager` | attack | Transformer attention 实现。部分 FedAvg/GPT2 脚本会改为 `eager`。 |
| `--precision` | `full`; `8bit` / `half` / `full` / `double` | attack | 模型/计算精度。 |
| `--hidden_act` | `None` | attack | rebuttal/消融用，覆盖模型激活函数。 |
| `--loss` | `ce`; `ce` / `mse` | attack | 攻击/重构使用的 loss 类型。 |

## 4. DAGER 攻击与重构超参数

| 参数 | 默认值 / 取值 | 所属入口 | 含义 |
|---|---:|---|---|
| `--parallel` | `1000` | attack | 重构候选并行处理规模。 |
| `--grad_b` | `None` | attack | 指定用于攻击的梯度 batch 维度；为空时按当前 batch 推断。 |
| `--n_layers` | `2` | attack | DAGER 用于 span/rank 检查的层数。 |
| `--rank_tol` | `None` | attack | 数值 rank 判定容差；某些脚本手动设为 `5e-6`、`5e-9` 等。 |
| `--rank_cutoff` | `20` | attack | rank 搜索/截断上限。 |
| `--l1_span_thresh` | `1e-5` | attack | L1 阶段 span 判定阈值。 |
| `--l2_span_thresh` | `1e-3` | attack | L2 阶段 span 判定阈值。 |
| `--l1_filter` | 必填；`maxB` / `all` | attack | L1 候选过滤策略。baseline 脚本通常用 `all`；BERT wrapper 用 `maxB`。 |
| `--l2_filter` | 必填；`overlap` / `non-overlap` | attack | L2 候选过滤策略；主实验常用 `non-overlap`。 |
| `--distinct_thresh` | `0.7` | attack | 候选去重/差异性阈值。 |
| `--max_ids` | `-1` | attack | 限制 token 候选数量；`-1` 表示不显式限制。partial non-prefix 中若大于 0 会覆盖 candidate cap。 |
| `--maxC` | `10000000` | attack | 候选组合搜索上限。 |
| `--reduce_incorrect` | `0` | attack | 是否减少错误候选的启发式开关。 |
| `--n_incorrect` | `None` -> `batch_size` | attack | 允许/处理的错误候选数量。 |
| `--max_len` | `1e10` | attack | 最大序列长度限制。 |
| `--p1_std_thrs` | `5` | attack | 噪声/异常值解码中第一阶段标准差阈值。 |
| `--l2_std_thrs` | `5` | attack | 噪声/异常值解码中第二阶段标准差阈值。 |
| `--dp_l2_filter` | `maxB`; `maxB` / `outliers` | attack | 带噪声情况下的 L2 过滤策略。 |
| `--dist_norm` | `l2`; `l1` / `l2` | attack | span 距离度量。 |
| `--grad_mode` | attack 默认 `eval`；train/proxy 默认 `train` | attack / train / proxy | 梯度计算时模型模式；影响 dropout 等训练态模块。 |

## 6. 防御 baseline 与通用防御参数

| 参数 / baseline | 默认值 / 取值 | 所属入口 | 含义 |
|---|---:|---|---|
| `--defense` | `none`; `none` / `noise` / `dpsgd` / `topk` / `compression` / `soteria` / `mixup` / `dager` / `lrb` / `lrbprojonly` | attack / train / proxy | 选择应用到客户端梯度或表示上的防御。 |
| `none` | - | all | 无防御，clean FedSGD/FedAvg 更新。 |
| `noise` | `--defense_noise` | all | 对已计算梯度逐元素加高斯噪声。 |
| `dpsgd` | `--defense_noise`, `--defense_clip_norm` | all | DP-SGD 风格：逐样本梯度裁剪、求均值、加噪；当前无 privacy accountant，不能当作正式 DP 保证。 |
| `topk` | `--defense_topk_ratio` | all | 每个 tensor 只保留绝对值最大的 top-k 比例，其余置零。 |
| `compression` | `--defense_n_bits` | all | QSGD 风格随机量化，按 tensor L2 norm 缩放。 |
| `soteria` | `--defense_soteria_pruning_rate` | all | Soteria 风格表示维度剪枝，按 classifier-input representation 敏感度打分并剪去最高分维度。 |
| `mixup` | `--defense_mixup_alpha` | all | 表示层 manifold MixUp 风格插值。`batch_size >= 2` 时更有意义。 |
| `dager` | `--defense_dager_*` | attack / train | 针对 DAGER 假设的动态基扰动、offset、切片、rank-limit 防御。 |
| `lrb` | `--defense_lrb_*` | all | Layer-wise Resolution Bottleneck，包含低分辨率投影、裁剪、残差噪声等组件。 |
| `lrbprojonly` | `--defense_lrb_keep_ratio_sensitive` | all | Projection-LRB / `proj_only` 顶层别名，只做低分辨率 signed-pool projection。 |
| `--defense_noise` | `None` | all | `noise` 的噪声标准差；`dpsgd` 的 noise multiplier `sigma`。 |
| `--defense_clip_norm` | `1.0` | all | `dpsgd` 逐样本 L2 裁剪阈值 `C`。 |
| `--defense_topk_ratio` | `0.1` | all | top-k 保留比例，范围 `(0, 1]`。 |
| `--defense_n_bits` | `8` | all | 梯度量化 bit 数，需 `>=1`。 |
| `--defense_soteria_pruning_rate` | `60.0` | all | Soteria 剪枝比例，百分数，范围 `[0, 100]`。 |
| `--defense_soteria_sample_dims` | `None` | all | Soteria 只随机评分部分 hidden dims，用于大模型降开销。 |
| `--defense_mixup_alpha` | `1.0` | all | MixUp 中 `Beta(alpha, alpha)` 的 alpha。 |
| `--defense_pct_mask` | `None` | all | 主防御后额外随机 mask 的比例；历史参数 `--pct_mask` 会映射到它。 |
| `--defense_rng_step` | `None` | all | 防御随机数 step/index；训练和 proxy 会逐 step 设置，用于每次更新新随机性。 |
| `--noise` | `None` | train / proxy | 旧兼容参数，自动映射到 `--defense_noise`。新实验建议统一用 `--defense ... --defense_noise ...`。 |
| `--pct_mask` | `None` | train / proxy | 旧兼容参数，自动映射到 `--defense_pct_mask`。 |

## 8. LRB / Projection-LRB 超参数

| 参数 / preset | 默认值 / 取值 | 所属入口 | 含义 |
|---|---:|---|---|
| `--defense_lrb_preset` | `custom` | all | LRB 命名 preset；非 `custom` 会覆盖底层 LRB 参数。 |
| `custom` | - | LRB preset | 保留显式传入的底层 LRB 参数。 |
| `identity_lrb` | - | LRB preset | 保留比例为 1，不裁剪不加噪；用于身份对照。 |
| `clip_only` | - | LRB preset | 只做层级裁剪，不做 projection/noise。 |
| `proj_only` | - | LRB preset | Projection-LRB 主体：只做 signed-pool 低分辨率投影。 |
| `lrbprojonly` | - | LRB preset | `--defense lrbprojonly` 会自动归一到该 preset。 |
| `proj_clip` | - | LRB preset | projection + clipping，不加残差噪声。 |
| `full_lrb` | - | LRB preset | projection + clipping + orthogonal residual noise。 |
| `pool_full` | - | LRB preset | 使用旧式 `pool` projection 的 full LRB。 |
| `rule_only` | - | LRB preset | full LRB，但敏感度校准只用规则先验。 |
| `empirical_only` | - | LRB preset | full LRB，但敏感度校准只用当前梯度统计。 |
| `uniform_all_sensitive` | - | LRB preset | 全层按 sensitive 层同样强度处理。 |
| `proj_rule_only` | - | LRB preset | projection-only，规则先验校准。 |
| `proj_empirical_only` | - | LRB preset | projection-only，经验统计校准。 |
| `proj_uniform` | - | LRB preset | projection-only，所有层使用同一 keep ratio。 |
| `proj_no_empirical` | - | LRB preset | projection-only，不使用经验校准。 |
| `--defense_lrb_sensitive_n_layers` | `2` | all | 前多少个 transformer 层视为敏感层，应用更强保护。 |
| `--defense_lrb_keep_ratio_sensitive` | `0.2` | all | 敏感层低分辨率保留比例；Projection-LRB 的核心 `k`。 |
| `--defense_lrb_keep_ratio_other` | `0.75` | all | 非敏感层保留比例。 |
| `--defense_lrb_clip_scale_sensitive` | `0.5` | all | 敏感层裁剪阈值相对中位梯度范数的倍率。 |
| `--defense_lrb_clip_scale_other` | `1.0` | all | 非敏感层裁剪阈值倍率。 |
| `--defense_lrb_noise_sensitive` | `0.03` | all | 敏感层残差正交噪声倍率。 |
| `--defense_lrb_noise_other` | `0.005` | all | 非敏感层残差正交噪声倍率。 |
| `--defense_lrb_empirical_weight` | `0.6` | all | LRB 敏感度校准的经验统计权重；`0` 为 rule-only，`1` 为 empirical-only。 |
| `--defense_lrb_calibration_samples` | `4096` | all | 每个 tensor 用于校准 sketch 的最大元素数。 |
| `--defense_lrb_projection` | `signed_pool`; `signed_pool` / `pool` | all | LRB 公共子空间投影方式；`signed_pool` 是当前默认。 |
